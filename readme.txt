This plugin gets book cover from Ozon.ru, saves the picture and changes post, adding the cover. Uses meta ozon_book_id.

== Description ==

Special plugin for Russian book lovers. 
It takes meta parameter `ozon_book_id` from post and when you save, the plugin get book cover picture from Ozon.ru. 
Your post is updated. So you don't need to fix neither your feed, nor cross-posting.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `ozon_book_cover.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Try to post. Don't forget to use meta tag `ozon_book_id`
